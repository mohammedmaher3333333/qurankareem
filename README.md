# qurankareem

## Getting Started

# Developed an app to assist users in listening to and memorizing the Holy Quran. Users can stream Quranic verses, mark favorite ayahs, and share them
### Tools
-	Architecture: MVVM    
-	State Management: Bloc   
-	API Integration: Dio
